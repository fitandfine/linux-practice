
These are the passwords as I have figured out in the process of climbing Bandit levels.
Instead of just making a text file of passwords, I want to make note of hints for
commands as well for not so obvious solutions ( from my current state )


level 0 password means the password required for ssh login to enter level0 i.e bandit0.
level 1 password means the password required for ssh login of user bandit1 and so on.

 
Remember to specify port 2220 ( or as mentioned in the website)
 or else it will use port 22 which is not desirable in this case. 
Open your terminal and get started.
 $: ssh bandit0@bandit.labs.overthewire.org -p 2220
 ( notice that -p 2220 part for specifying ssh connection on port 2220)
